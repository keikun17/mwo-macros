Mechwarrior Online Macros
=======================

### uac5x3.ahk

binds Mouse1 to `4,5,6,3----<until release>`

#### Usage

`<F-12>` to enable Macro

Map UACs to single fire on `4,5,6` and group fire on `3`

Remap keys to


    Command     Key Map   Alt Key Map
    FWG1        1         Mouse1
    FWG2        2         Mouse2
    FWG3        3         Mouse5
    FWG4        4         Mouse4
    FWG5        5
    FWG6        6
